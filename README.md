# Formulario de Inscripción - Taller de Desarrollo Web

## Descripción

Este proyecto consiste en la creación de un formulario HTML para registrar estudiantes interesados en participar en un taller de desarrollo web.

El formulario fue desarrollado únicamente con HTML, sin utilizar CSS ni JavaScript, siguiendo las instrucciones de la actividad académica.

## Contenido del formulario

El formulario incluye las siguientes secciones:

* Datos personales
* Datos académicos
* Preferencias del taller
* Disponibilidad
* Información adicional

## Elementos HTML utilizados

En el desarrollo del formulario se utilizaron las siguientes etiquetas HTML:

* `<form>`
* `<label>`
* `<input>`
* `<select>`
* `<option>`
* `<textarea>`
* `<button>`
* `<fieldset>`
* `<legend>`

Además, se aplicaron atributos de validación HTML como:

* `required`
* `placeholder`
* `min`
* `max`
* `maxlength`
* `minlength`
* `pattern`
* `step`
